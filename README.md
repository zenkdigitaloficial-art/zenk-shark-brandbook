# ZENK / Shark OS Brandbook

Brandbook oficial em HTML para orientar design, IA e desenvolvimento do Shark OS.

## Arquivos

- `index.html` — brandbook navegavel, com HTML/CSS/JS inline, pronto para GitHub Pages.

## Direcao

- Interface clara em branco/off-white.
- Azul ZENK como cor primaria.
- Verde para sucesso/progresso.
- Vermelho para risco/atraso.
- Dourado para XP, moedas e recompensas.
- Fontes oficiais: Syne + Inter.

## Uso

Use este brandbook como fonte fixa de contexto antes de criar telas, componentes, prompts ou redesigns do Shark OS.
